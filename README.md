## 🛑 Technical Constraints
To prove my grasp of core programming concepts, this project was built under strict constraints:
* **Allowed:** Python fundamentals (loops, dicts, strings, f-strings, datetime), and NumPy (for the heatmap matrix).
* **Forbidden:** pandas (no dataframes), matplotlib/seaborn (all visuals are text-based ASCII), re (regex), and NLP libraries. 

## ⚙️ How to Run
1. Download the `GroupDNA_Samarpan.ipynb` file and `hostel_bois.txt`.
2. Open the notebook in Google Colab or Jupyter Notebook.
3. Upload `hostel_bois.txt` to the environment.
4. Run all cells from top to bottom to generate the final report.
